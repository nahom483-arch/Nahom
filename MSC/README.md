# Ball and Beam – (Work in Progress!)

Control project for a ball‑and‑beam system.  
Goal: Regulate the ball position (l) by adjusting the beam angle (φ).

## Current Status (ongoing):

- Degrees of freedom identified
- Lagrange equation derived
- Static equilibrium analysis
- Static characteristic plot (equilibrium points)
- Block diagram of the dynamics
